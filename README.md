# Nightreign Reforged

**NOTE:** No relation to [Elden Ring Reforged](https://www.nexusmods.com/eldenring/mods/541)

## About

Collection of mods for ER Nightreign. Includes dll mods and merged regulation.bin.

## Installation

1. Clone or download this repository anywhere on your PC
2. Unzip archive (if downloaded zip)
3. Setup `nrsc_settings.ini`: copy `SCNR/nrsc_settings.ini.example` to `SCNR/nrsc_settings.ini`

**NOTE:** You will need to make edits to your ME3 profile if you have used the Randomizer mod.
